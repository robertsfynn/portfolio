# Portfolio

My portfolio written in Gatsby

[Demo](http://robertsfynn.dev/)

## Getting Started

### Installing

npm:

```
npm install
npm start
```

yarn:

```
yarn install
yarn start
```

## Built With

- [Gatsby](https://www.gatsbyjs.org/)
- [Styled Components](https://www.styled-components.com/)
- [React Spring](https://www.react-spring.io/)
